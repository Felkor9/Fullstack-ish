CDN
Ett Content Delivery Network är ett nätverk av servrar som är utspridda över hela världen för att snabba upp leveransen av webbsidor och digitalt innehåll. Genom att lagra kopior av filer, som bilder, videor och webbsidor, på flera servrar kan innehållet levereras från den server som ligger närmast användaren. Detta minskar laddningstider och förbättrar användarupplevelsen. Ett CDN ökar även tillgänglighet, hanterar stora trafikmängder och skyddar mot attacker som DDoS.

Monitoring
Monitorering handlar om att kontinuerligt följa upp och analysera ett systems prestanda, hälsa och tillgänglighet. Genom att samla in data som svarstider, fel, trafikvolym och resursanvändning kan man snabbt upptäcka problem och förebygga driftstörningar. Det gör det möjligt att larma vid avvikelser, identifiera flaskhalsar och analysera orsaker till fel, vilket förbättrar både stabilitet och användarupplevelse.
